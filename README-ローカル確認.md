# ローカルでLPを確認する方法

フォルダ名に日本語（スピ賃）が含まれるため、ターミナルでパスを指定しての起動が失敗することがあります。以下のいずれかで起動してください。

## 方法1: 起動用スクリプトを使う（推奨）

1. **エクスプローラー**でこのフォルダ `LP(スピ賃)` を開く
2. **start.bat** をダブルクリック  
   → コマンドプロンプトが開き、サーバーが起動します
3. ブラウザで **http://localhost:5500/** を開く
4. 終了するときは、コマンドプロンプトで **Ctrl+C**

PowerShell を使う場合:
- このフォルダで **start.ps1** を右クリック → **「PowerShellで実行」**
- または、このフォルダでターミナルを開き `.\start.ps1` を実行

## 方法2: このフォルダを開いた状態でコマンドを打つ

1. エクスプローラーで **LP(スピ賃)** を開く
2. アドレス欄をクリックしてパスを選択し、`cmd` と入力して Enter  
   → このフォルダをカレントにしたコマンドプロンプトが開く
3. 次のどちらかを実行:
   - `python -m http.server 5500`
   - Python が入っていない場合: `py -m http.server 5500`
4. ブラウザで **http://localhost:5500/** を開く

## 方法3: Cursor / VS Code の Live Server

1. 拡張機能 **Live Server** をインストール
2. **index.html** を開いた状態で、エディタ右下の **「Go Live」** をクリック  
   → ブラウザで自動的に開きます

---

- ポート 5500 が別のソフトで使われている場合は、別の番号（例: 8080）に変更してください。  
  - start.ps1: `$port = 8080` に変更  
  - start.bat: `5500` を `8080` に変更  
  - 手動: `python -m http.server 8080`
- アイコン画像は `assets/webapp-icon/icon.png` を置くとLPに表示されます（未配置の場合は「APP」と表示されます）。
